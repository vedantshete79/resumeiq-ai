## 🚀 Overview

**ResumeIQ AI** is a SaaS-style ATS Resume Analyzer and Career Optimization Suite designed with an **AI Cyber Dashboard** theme.

### Key Features
- 🎯 **ATS Score & Match Engine**: Calculates 0-100 ATS compatibility score using TF-IDF vector analytics and Cosine similarity.
- 🟢🔴🔵 **Keyword Breakdown**: Categorizes matched, missing, and AI-suggested terms.
- ⚡ **Section Diagnostics**: Evaluates 7 core resume sections (Contact Info, Summary, Skills, Experience, Education, Projects, Certifications).
- 📝 **AI Resume Builder**: Interactive live resume builder with instant PDF export.
- ✉️ **AI Cover Letter Generator**: Custom cover letter builder tailored to target roles & company names.
- 💼 **LinkedIn Profile Analyzer & Interview Prep**: Evaluates search visibility and generates role-tailored technical/behavioral interview questions.

## ⚙️ Quick Start Guide

### 1. Clone the Repository
```bash
git clone https://github.com/YOUR_USERNAME/resumeiq-ai.git
cd resumeiq-ai
```

### 2. Run the Application
```bash
python server.py
```

### 3. Open in Browser
Navigate to `http://localhost:8000` in your web browser.
